# Decision Architecture

## Job Fit Criteria

The LLM evaluates each role using:

- AI relevance
- Skill match
- Growth potential
- Portfolio alignment
- Practical probability of getting interviews
- Alignment with AI Engineer, GenAI Engineer, or AI Application Engineer goals

## Recommendations

The app asks the LLM to produce one of three recommendations:

- Apply
- Maybe
- Skip

## Output Design

Outputs are structured in copy-ready Markdown sections so the user can reuse analysis, resume bullets, and messages directly.
