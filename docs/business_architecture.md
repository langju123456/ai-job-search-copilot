# Business Architecture

## User

The target user is an early-career job seeker pursuing AI Engineer, GenAI Engineer, or AI Application Engineer roles.

## Job To Be Done

The user needs to decide which jobs are worth applying to, tailor their resume quickly, write credible networking messages, and track application activity.

## MVP Value

The app reduces repetitive job-search work by converting a resume and job description into specific application strategy outputs.

## Core Workflow

1. Paste user profile or resume.
2. Paste job description.
3. Generate job-fit analysis.
4. Generate resume tailoring.
5. Generate networking messages.
6. Save the application.
7. Review saved applications in the tracker.
